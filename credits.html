Credits

Credits to open source components used in the development of our product

Website

Using Jekyll

Jekyll is an Open-Source static site generator by GitHub. Jekyll provided many features which helped us to modularize the website such as a looping facility which we could use to render common HTML through a loop rather than copying it out many times. This was especially useful in the Meet the team page as it allowed us to write a single div which contained the semantics for each team member then have the generator generate the page using content provided in markdown files.

SASS and LineageOS

We decided to use SASS over CSS. SASS is built upon CSS but also adds useful tools such as variables to hold common values such as brand colors and fonts etc. The SASS that has been implemented was adapted from the Open-Source LineageOS website found here.These SASS components were adapted by LineageOS from the Bootstrap Framework and Material Kit, all of which are licensed under the permissive MIT license. Also, from the LineageOS website, we adapted their blog page, 404 Page and we have also adapted some of their layouts for other pages in our website as we found it to meet our aesthetic requirements and it also provided an efficient way to write blog posts.
